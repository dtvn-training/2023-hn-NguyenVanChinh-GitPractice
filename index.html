Features of Git

  • History tracking: Keeps track of code changes
  • Collaboration: Allows you to share and co-edit files with others.
  • Branching: Allows you to have multiple branches that can be independent of each other.
  • Free and open source: You can use Git for free. You can also modify the software to improve it.
  • Distributed:
    In a Distributed Version Control System, users don’t just check out the latest version of the source code; rather, they fully mirror the repo, including its full history. If any server dies, any of the local repositories can be copied back up to the server to restore it. Doesn’t require an internet connection, so most development, except pushing and pulling, can be done offline. Developers can continue working when losing internet connection.
